# 과제2: 외부 라이브러리(requests) 활용 프로그램

이 프로젝트는 파이썬의 대표적인 HTTP 요청 라이브러리인 `requests`를 활용하여 외부 기상 데이터 API로부터 실시간 서울 날씨 정보를 수집하고 정돈하여 출력하는 프로그램입니다.

---

##  1. 사용한 외부 라이브러리 정보

* **라이브러리 이름**: `requests`
* **설치된 버전**: `2.32.3` 
* **선택 및 버전 이유**:
  - **선택 이유**: 파이썬 내장 모듈인 `urllib`보다 문법이 훨씬 직관적이고 가독성이 높습니다. 웹 API 통신 및 데이터 파싱 과정을 가장 효율적으로 구현할 수 있는 실무 표준 라이브러리이기 때문에 선택했습니다.
  - **버전 선택 이유**: 보안 취약점이 해결되고 가장 안정적으로 작동하는 최신 LTS(장기 지원) 기반의 버전을 선택하여 설치했습니다.

---
